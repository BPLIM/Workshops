# BPLIM Workshop 2025

In this folder you can find all the materials from the Workshop on "Speeding Up Empirical Research:
Tools and Techniques for Fast Computing". Recordings of all sessions are available [here](https://www.bportugal.pt/evento/workshop-speeding-empirical-research-tools-and-techniques-fast-computing-bplim).
